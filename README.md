 < Sistema de Autenticação >

Projeto acadêmico desenvolvido no curso de **Sistemas de Informação**,
com foco em organização de código, boas práticas, versionamento 
e aplicações conceituais de padrão de projetos.

 ---- Descrição -----
Sistema responsável por autenticação de usuários, estruturado em
**backend** e **frontend**, aplicando Programação Orientada a Objetos
e padrões de projeto. Para o desenvolvimento do trabalho foi utilizado o framework "Flask".

 ---- Tecnologias e Conceitos -----
- Python
- Flask
- Arquitetura em camadas
- Programação Orientada a Objetos (POO)
- Padrões de Projeto (Strategy)
- Git e GitHub


--------- > Como executar o backend < ---------

// No terminal 
1 python -m venv venv

2 venv\Scripts\activate

3 pip install -r requirements.txt

4 python login_gui_corpotate.py
